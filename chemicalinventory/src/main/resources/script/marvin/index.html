Files

  • chemaxon/ - Java classes loaded at runtime
  • chemaxon/marvin/templates/ - template molecules
  • marvin.jar - JAR file containing the Marvin/AWT applets
  • jmarvin.jar - signed JAR file containing the Marvin/Swing applets
  • sjars - signed modules loaded at runtime for Marvin/Swing applets
  • xjars - extra resources for Calculator Plugins
  • mols-2d/, mols-3d/ - molecules used in the examples
  • mspace.jar - JAR file containing the MarvinSpace applet
  • jogl.jar - Java binding for OpenGL (required by MarvinSpace)
  • jogl.dll - Windows native library for jogl.jar
  • libjogl.so - Linux native library for jogl.jar
  • libjogl.jnilib - OS X native library for jogl.jar
  • inchi.dll - Windows native library for IUPAC InChi import/export
  • libinchi.so - Linux native library for IUPAC InChi import/export
  • libinchi.jnilib - OS X native library for IUPAC InChi import/export
