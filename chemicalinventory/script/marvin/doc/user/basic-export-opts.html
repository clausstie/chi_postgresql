Basic export options

a or +a     Aromatize (level 1). Example: "XXX:a"
a_day or +a_daylight   Daylight compatible aromatization (level 2). Example: "XXX:a_day"
-a Dearomatize. Example: "XXX:-a"
H or +H Add explicit Hydrogen atoms. Example: "XXX:H"
-H Remove explicit Hydrogen atoms. Example: "XXX:-H"
Here, XXX can be any molecule or image format like mol, sdf, rdf, rgf, rxn, csmol, cssdf, csrdf, csrgf, csrxn, smiles, cxsmiles, mrv, cml, jpeg, png, ppm or svg, but aromatization options have no effect on formats which do not store bond orders like cube, pdb and xyz.
